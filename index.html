Description

MorphDOM moves HTML elements depending on the window's width.

Dependencies

  • Browser.js
  • Device.js
  • underscore - http://underscorejs.org/
  • jQuery - http://jquery.com/

Usage

The configuration for MorphDOM is handled by data-attributes in your HTML. Each element that will be moved with MorphDOM needs it's own configuration.

Objects are placed in groups, all members of each group are moved into the same HTML Element specified at instantiation in JS

data-attributes

data-morph*
Name of the morph group
data-morph-devices*
A comma separated list of devices in which the element shall be moved. ej. "mobile, tablet"
data-morph-stack
Position of the element
data-morph-type
The type of morph that should be applyed to the element. The available options are "clone" and "copy" (default:"copy"). Elements with morph type clone are cloned instead of moved
data-morph-target
CSS selector specifying a different target for the current element only
* Required

After configuring your HTML elements you need to create an instance of MorphDOM for each group.

Options

rel*
Name of the morph group
target*
CSS selector specifying the morph target for this group
morphs
Object containing callbacks for each device. Currently the available callbacks are "init" and "destroy". Each device can have it's own callbacks , callbacks can be shared between devices by indicating the name of the device from which you want to execute the callback.
ej. 
morphs:{
  tablet:{
    init:function(device, current_device){
      console.log("Entering tablet");
    },
    destroy:function(){
      console.log("Exiting tablet");
    }
  },
  mobile:{
    init:function(device, current_device){
      console.log("Entering mobile");
    },
    destroy:"tablet"
  }
}
* Required

Demo

This paragraph will move to #mobile_container when the window's width is less than or equals 767px


HTML

<p
  data-morph="subnav" 
  data-morph-stack="1"
  data-morph-devices="mobile">
    This paragraph will move to #mobile_container when 
    the window's width is less than or equals 767px
</p>

JS

var subnav = new TCE.MorphDOM({
  rel: "subnav",
  target: "#mobile_container",
  morphs:{
    tablet:{
      init:function(device, current_device){
        console.log("Entering tablet");
      },
      destroy:function(){
        console.log("Exiting tablet");
      }
    },
    mobile:{
      init:function(device, current_device){
        console.log("Entering mobile");
      },
      destroy:function(){
        console.log("Exiting mobile");
      }
    }
  }
});
